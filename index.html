Extension Simulator: Mermaid Renderer

This demo shows how the Chrome extension detects Mermaid code blocks on any webpage and adds a render button to each block. Click the buttons to see diagrams rendered inline using Mermaid v11.

Sample Webpage Content

Simple Flowchart

graph TD
    A[Start] --> B{Decision}
    B -->|Yes| C[Action 1]
    B -->|No| D[Action 2]
    C --> E[End]
    D --> E
